Visualizer Script (heatmap_viz.py).
This script reads the kernel_matrix.csv generated by your engine and turns it into a color-coded "Theological Topology" map.
The Visualizer: heatmap_viz.py
You will need matplotlib and numpy installed (pip install matplotlib numpy).
import matplotlib.pyplot as plt
import numpy as np
import csv

# ==========================================
# CONFIGURATION
# ==========================================

# The 18 Anchor Labels (Must match the order in kernel_monolith.py)
LABELS = [
    "Gen 1:27 (Image)",
    "Gen 2:18 (Helper)",
    "Gen 3:16 (Rule)",
    "Eph 5:21 (Submit)",
    "Rom 3:23 (Sin)",
    "Rom 9:13 (Election)",
    "Jas 2:24 (Works)",
    "Gal 3:28 (Equality)",
    "Eph 1:7 (Grace)",
    "Rev 6:15 (Wrath)",
    "Matt 23:27 (Hypocrisy)",
    "Exod 21:2 (Hebrew)",
    "Lev 25:44 (Slave)",
    "Philemon (Brother)",
    "John 14:6 (The Way)",
    "Acts 4:12 (The Name)",
    "Matt 7:13 (Narrow)",
    "Matt 25:41 (Fire)"
]

INPUT_FILE = "kernel_matrix.csv"
OUTPUT_IMAGE = "theological_heatmap.png"

# ==========================================
# GENERATE HEATMAP
# ==========================================

def generate_heatmap():
    print(f"[*] Reading {INPUT_FILE}...")
    
    try:
        # Load the 18x18 matrix
        matrix = np.loadtxt(INPUT_FILE, delimiter=',')
    except FileNotFoundError:
        print(f"[!] Error: Could not find {INPUT_FILE}. Run the engine first!")
        return

    # Setup the Plot
    fig, ax = plt.subplots(figsize=(12, 10))
    
    # Create the Heatmap
    # 'coolwarm' colormap: Blue (Low Correlation) -> Red (High Correlation)
    im = ax.imshow(matrix, cmap='coolwarm', interpolation='nearest')

    # Add Colorbar
    cbar = ax.figure.colorbar(im, ax=ax)
    cbar.ax.set_ylabel("Resonance Strength (0.0 - 1.0)", rotation=-90, va="bottom")

    # Configure Axis Labels
    ax.set_xticks(np.arange(len(LABELS)))
    ax.set_yticks(np.arange(len(LABELS)))
    ax.set_xticklabels(LABELS, rotation=45, ha="right", fontsize=9)
    ax.set_yticklabels(LABELS, fontsize=9)

    # Add Text Annotations (Optional: Show values in the squares)
    # Loop over data dimensions and create text annotations.
    for i in range(len(LABELS)):
        for j in range(len(LABELS)):
            text = ax.text(j, i, f"{matrix[i, j]:.2f}",
                           ha="center", va="center", color="black", fontsize=6)

    # Title and Layout
    ax.set_title("Kernel Matrix: Theological Tension & Resonance")
    plt.tight_layout()

    # Save to File
    plt.savefig(OUTPUT_IMAGE, dpi=300)
    print(f"[*] Heatmap saved to {OUTPUT_IMAGE}")
    plt.show()

if __name__ == "__main__":
    generate_heatmap()

How to Read the Map
Once you run this, you will see a grid of colored squares. Here is how to interpret the theology from the colors:
 * Dark Red Squares (Hot):
   * These two verses represent the same doctrinal pole.
   * Example: You should see a red intersection between Gen 1:27 and Gen 2:18 (Creation/Gender affinity).
 * Dark Blue Squares (Cold):
   * These two verses are theologically distant or opposing.
   * Example: You might see a blue intersection between Gal 3:28 (Equality) and Lev 25:44 (Slavery).
 * The Diagonal Line:
   * You will see a solid red line running from top-left to bottom-right. This is normal; every verse matches itself 100%.
